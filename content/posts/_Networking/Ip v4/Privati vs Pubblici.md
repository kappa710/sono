---
title : "Privati vs Pubblici"
---
Gli indirzzi ipv4 si dividono in Privati e Pubblici

## IP PUBBLICI

| CLASSE | RANGE   | DIFFERENZA | TIPO          |
| ------ | ------- | ---------- | ------------- |
| A      | 1-126   | +128       | grandi reti   |
| B      | 128-191 | +64        | reti medie    |
| C      | 192-223 | +32        | reti piccole  |
| D      | 224-239 | +16        | multicast     |
| E      | 240-255 | +16        | speriementali |


## IP PRIVATI

Indirizzi privati per convenzione fanno parta di queste subnet

| CLASSE | RANGE IP       |
| ------ | -------------- |
| A      | 10.0.0.0/8     |
| B      | 172.16.0.0/12- |
| C      | 192.168.0.0/16 |
