`ipv6 unicast-routing`

vai in interfaccia 
`ipv6 address ip/subenet`
per generare automaticamente interface host da mac address interfaccia
`ipv6 address 2001:db8::/64 eui-64`

per visualizzare interfacce con ipv6
`show ipv6 interface brief`

per anycast
`ip address ip/subnet anycast`
